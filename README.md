# Introducccio_a_la_Probabilitat
Resolució de problemes bàsics probabilistics a través de l'aplicació de la definició freqüèntista de Laplace. El repositori es divideix en: 

  1. Practica 1 - Simulació i obtenció de probabilitats aplicant mètode de MonteCarlo
  2. Practica 2 - Resolució de problemes
  3. Treball en grup - Conté l'algoritme de simulació de cerca d'un individu perdut degut i la probabilitat de trobar aquell individua través de la simulació i certes condicions de cerca.
